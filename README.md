# База знаний

Это сборник инструкций и заметок на разнообразные темы.

Опубликованная версия — <https://romanvolodin.github.io/howto/>

## Запуск сервера с документацией

В проекте используется [пакетный менеджер uv](https://docs.astral.sh/uv/) для управления виртуальным окружением и зависимостями.

### Установите `uv`

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

### Запустите сервер MkDocs

```bash
uv run mkdocs serve
```

Команда делает следующее:

- создаёт виртуальное окружение в папке `.venv`
- активирует его
- устанавливает зависимости
- и запускает сервер

По умолчанию сервер будет запущен на порту 8000.

Задать другой порт можно параметром `--dev-addr`:

```bash
uv run mkdocs serve --dev-addr 0.0.0.0:3000
```
