# Как скопировать ключ на сервер

У меня есть логин и пароль от сервера, но я хочу подключать по ключу. Как это сделать?

## Сгенерируйте SSH-ключ (если ещё нет)

```bash
ssh-keygen -t ed25519
```

## Скопируйте публичный ключ на сервер

```bash
ssh-copy-id user@server
```

## Подключайтесь без пароля

```bash
ssh user@server
```

Для большего удобства [настройте алиас](ssh-alias.md).
